What is React

React is a library for JS which was develop by Facebook, it can manage and product the UI, but it is not a framework.

Characteristic of React

1.用純 JS 在前端產生 HTML （一般來說是在後端產生 HTML 送到前端）
2.使用虛擬 DOM （Virtual DOM），重繪時效率高
3.自定義 部件（Component），方便開發
4.父子 Component 可透過 props 通訊
5.只負責 MVC 的 View 部份，所以不算框架，彈性高
6.因為完全是 JS 操作 UI 的關係，使得它可以跟後端分離，達到即時互動、自動更新的效果
7.只是一個 JS 函式庫，所以容量小易移植
